Installation

recops developed on x64 Linux box using Python version 3.10.2. Is expected to work flawlessly on every linux box with Python versions above 3.7. In order to run on other systems we suggest to use the Dockerized version.

Functionality

In this page we walk through recops core design and functionality. recops is mainly a command-line interface (CLI) toolkit and well documented, use --help to get description of each function and the available options.
